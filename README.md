⚡ SparkGuard – Backend
SparkGuard is an intelligent electricity theft detection system.
The backend provides secure RESTful APIs for authentication, data management, and theft detection using
deep learning model.

🛠 Tech Stack

Node.js

Express.js

MongoDB

Mongoose

JWT Authentication

⚙️ Run Code:
npm install
npm run start-dev
